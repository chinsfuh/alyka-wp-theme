# Alyka WordPress Theme

A modern, lightweight WordPress theme designed for flexibility and ease of use.

## Installation

1. Download the theme files
2. Upload to `wp-content/themes/alyka/`
3. Activate the theme in WordPress Admin

## Development

Clone the repository:
```bash
git clone https://github.com/chinsfuh/alyka-wp-theme.git
```

## Requirements

- WordPress 5.0 or higher
- PHP 7.2 or higher
